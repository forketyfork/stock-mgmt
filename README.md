## A warehouse management application

### Tools
* Java 8
* Grails 3.2.8
* Gradle 3.4.1
* H2 database

### Building
```
./gradlew assembly`
```

### Running
```
./gradlew bootRun`
```

### Users
User `admin` with password `777` has complete access to the application.
User `manager` with password `999` has read-only access.
